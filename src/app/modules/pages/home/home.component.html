Welcome to my

ANGULAR NGRX MongoDB

Application!

This is an Angular-NGRX based Application.

In the project building I am using "Oopcoders.com" tutorial, but I have made significant modifications to the original materials. For the backend I use Node express, MongoDB, seeder etc. instead of a fake json-server. The frontend is created by using Angular 16, NGRX tools, Bootstrap, Spinner and Alert effects etc.
I have customized shopping cart procedure (cart items should be saved to the user data) and place order procedure (an order creation triggers update of the shopping cart and reduction of the ordered items quantity in the products stock). Furthermore, I have added some features, for example user registration, ticket request, user profile, mailbox and notification e-mail, product review and rating, order return, form validation, responsive navbar, password reset etc.
Some another of my ideas in this project:
- Premium Membership (free shipping and loyalty points voucher): once the fee has been paid, the user status should be updated automatically and displayed in the user profile, in place order procedure. Manager area allows to generate coupon (loyalty points voucher). Voucher discount will be displayed in order summary by coupon/voucher submission.
- Franchise Membership: once the fee has been paid, user should get access to the test completion, which should be automatically validated and the result should be displayed, etc.
